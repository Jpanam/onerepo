Como obter xPub no Ubuntu?

Siga o passo-a-passo para obter a chave xPub em sua KeepKey utilizando Ubuntu:

I. Instale o pip

- Abra o terminal
- Digite o seguinte comando: - sudo easy_install pip

II. Instale o Cython

- Abra o terminal
- Digite o seguinte comando: - sudo pip install cython

III. Install a Libsub

- Abra o terminal
- Digite o seguinte comando: - sudo pip install libusb-1.0.0-dev

IV. Instale o KeepKey Python Client

- Abra o terminal
- Digite o seguinte comando: - sudo pip install HTTPS://GITHUB.COM/KEEPKEY/PYTHON-KEEPKEY/ARCHIVE/MASTER.ZIP


V. Obtenha sua chave xpub 

- Abra o terminal
- Digite o seguinte comando: - keepkeyctl get_public_node -n "44'/0'/0'"
- Digite o seu PIN no terminal de acordo com os números exibidos na tela de sua KeepKey
